# RazeArc

RazeArc is a retro-style arena FPS where speed, aggression, and movement mastery decide every fight.

Drop into a low-poly combat zone, carve through enemy packs, and push from one arena to the next with a growing arsenal. Each level raises the pressure with tighter resources, harsher hazards, and heavier resistance until the final showdown.

## What It Feels Like

- Fast, old-school FPS momentum with modern responsiveness
- Weapon variety from close-quarters melee to high-impact rockets
- Vertical movement and explosive repositioning that reward confidence
- Arena encounters that force quick target priority and smart pathing
- Retro visual direction with straightforward, high-contrast combat readability

## Current Experience

The current prototype includes a full 3-level run with:

- Progressive arenas and level-to-level advancement
- Multiple weapon pickups and ammo/health recovery loops
- Environmental threats and arena-lock combat moments
- Win/lose flow and core HUD feedback throughout play

RazeArc is being built as a focused, replayable FPS prototype: quick to start, easy to read, and satisfying to push harder each run.
